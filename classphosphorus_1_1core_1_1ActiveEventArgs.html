Detailed Description

Active Event arguments.

This is the type of EventArgs passed into all of your Active Events. To retrieve parameters passed into, or return values from your Active Events, use the Args property.

ActiveEventArgs phosphorus.core.ActiveEventArgs.Base
getset

Returns the base Active Event for the currently raised Active Event.

If your Active Event is overriding another Active Event, then this is the EventArgs for the base event implementation. If you wish to invoke your base Active Event, you can do so using ApplicationContent.CallBase, passing in this property.
