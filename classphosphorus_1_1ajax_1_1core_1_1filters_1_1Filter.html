Detailed Description

Base class for all http response filters in phosphorus.ajax.

The Filter class is used to override the response back to the client, to make sure we either return an HTML document containing the necessary JavaScript inclusions, or a JSON object during Ajax requests. Normally you do not need to fiddle with this class yourself, since the framework takes care of initializing instances of this, on a per need basis automatically. If you absolutely must access the Filter for your page, you can do so by accessing it through Page.Response.Filter.
