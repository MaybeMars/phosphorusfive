Detailed Description

Class wrapping your configuration section from your web.config that defines which assemblies to use as Active Event plugins.

To add your own plugins to your application pool, add one key for the assembly in your web.config at the appropriate place.

string phosphorus.applicationpool.code.ActiveEventAssemblies.PluginDirectory
get

Gets the plugin directory.

By default, Phosphorus will use your "bin" folder inside of your application pool to resolve its plugin asemblies. You can however override this behavior, by explicitly changing the folder it looks in by changing this property through your web.config file.
