Detailed Description

Wraps a single match.

When an Expression is evaluated, and returned as a Match object, then this class encapsulates one single matched item (node, value, path, name or count) from your Expression.

Match.MatchType phosphorus.expressions.MatchEntity.TypeOfMatch
get

Type of match.

Sometimes, especially when you're using reference expressions, then the type of match you end up with, is a mixed type. Meaning, some of your match entity items will be of type 'node', while others might be of type 'value', and so on. If so is the case, then this property will return the type declaration of your match entity item, instead of the type declaration of your Expression as a whole.

Type of match for match entity item.
