Detailed Description

Contains all core functionality in Phosphorus.Ajax.

Contains all the core helper and supporting features in Phosphorus.Ajax, such as the Manager, AjaxPage and so on. Helps glue together the Ajax Widgets with your Page, and helps you handle ViewState, create Ajax WebMethods and so on.
