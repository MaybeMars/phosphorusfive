Detailed Description

Responsible for serializing 'application/x-www-form-urlencode' types of requests.

This is the default serializer used, if no 'Content-Type' is declared for your HTTP POST or PUT requests. Basically it just URL-encodes the HTTP request, the same way a browser would for its form elements.
