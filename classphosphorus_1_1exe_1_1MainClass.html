Detailed Description

Main class for the pf.lambda console executor.

This is the main entry class for the "lambda.exe" console program, that allows for eexecuting Hyperlisp and pf.lambda code through the console, either directly using its "immediate mode", or indirectly through Hyperlisp files.

Class exposes two Active Events of its own, [pf.console.write] and [pf.console.write-line]. These two Active Events simply writes to the standard output, using the Console.Write/WriteLine from .Net and Mono.
