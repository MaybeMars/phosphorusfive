Detailed Description

Root iterators for nested IteratorGroup iterators.

Helper Iterator to declare a new IteratorGroup. Internally used when a new group, or sub-expression is declared, using parenthesis.

This iterator is never directly consumed in your code, but automatically created for you when you create an IteratorGroup Iterator.
