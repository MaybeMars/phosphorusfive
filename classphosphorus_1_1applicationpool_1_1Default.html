Detailed Description

Main .aspx web page for your Application Pool.

This is the main, and only, .aspx page for your Application Pool, from which all web requests are being handled. It contains many useful helper Active events, for things such as including JavaScript files, including StyleSheet files, creating and manipulating web widgets, etc.
