Detailed Description

Main namespace for everything regarding dynamically created Active Events.

A dynamic Active Event is created using the [event] keyword. You can also override dynamically created Active Events by using the [override] keyword. This namespace, contains the necessary Active Events to handle dynamically created Active Events.
