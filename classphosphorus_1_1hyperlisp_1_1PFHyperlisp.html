Detailed Description

Class to help transform between Hyperlisp and Nodes.

Contains the [pf.hyperlisp.hyperlisp2lambda] and the [pf.hyperlisp.lambda2hyperlisp] Active Events, necessary to be able to parse Hyperlisp from pf.lambda nodes, and vice versa.
